Point Comparison App

    Features:
  • Use the arrow keys to look around the grid.
  • Use "Z" to zoom in and "X" to zoom out.
  • Use the mouse pointer to get the x and y values at a particular point.
  • Press Enter to add a point. Specify a point title, x value, and y value by clicking in the input box that appears, typing a name/number, then clicking away and pressing Enter again or pressing Enter twice.
  • Press "L" to toggle a list which displays all entered points so far.
  • Press "H" to toggle hiding names written by each point. When hidden, only the name of a point being moused over will be shown.
  • Use "D" to delete a point being hovered over. Confirm by pressing "Y" or cancel by pressing "N".
  • While the list of points is being displayed, it can be navigated through by pressing "S" to go down and "W" to go up.
